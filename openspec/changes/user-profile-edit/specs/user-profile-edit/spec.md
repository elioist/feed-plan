## ADDED Requirements

### Requirement: 用户头像上传

系统 SHALL 支持用户上传头像图片。头像 MUST 为 1:1 正方形比例，支持 JPG、PNG、WebP 格式。上传时 MUST 提供裁剪功能，裁剪后保存到服务器。

#### Scenario: 上传头像成功
- **WHEN** 用户选择图片并完成裁剪后上传
- **THEN** 系统保存头像到服务器，返回头像路径，更新用户头像字段

#### Scenario: 上传非图片文件
- **WHEN** 用户上传非图片文件（如 PDF、TXT）
- **THEN** 系统拒绝上传并提示"仅支持图片格式"

### Requirement: 用户名修改

系统 SHALL 支持修改用户名，修改时 MUST 校验唯一性。新用户名不可与已有用户名重复。

#### Scenario: 修改用户名成功
- **WHEN** 用户提交不存在的新用户名
- **THEN** 系统更新用户名，后续登录使用新用户名

#### Scenario: 用户名已存在
- **WHEN** 用户提交已存在的用户名
- **THEN** 系统返回 409 错误，提示"用户名已存在"

#### Scenario: 用户名为空
- **WHEN** 用户提交空用户名
- **THEN** 系统返回 400 错误，提示"用户名不能为空"
