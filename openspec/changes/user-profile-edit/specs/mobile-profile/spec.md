## ADDED Requirements

### Requirement: 移动端个人页显示头像

系统 SHALL 在移动端个人页显示用户头像。用户 SHALL 能够点击头像修改自己的头像。头像 MUST 支持 1:1 裁剪上传。

#### Scenario: 查看个人页头像
- **WHEN** 用户进入个人页
- **THEN** 系统显示用户头像（有头像时显示图片，无头像时显示用户名首字）

#### Scenario: 修改头像
- **WHEN** 用户点击头像并选择新图片完成裁剪
- **THEN** 系统上传头像，更新显示，展示成功反馈

#### Scenario: 无头像显示默认样式
- **WHEN** 用户未上传头像
- **THEN** 系统显示用户名首字作为默认头像
